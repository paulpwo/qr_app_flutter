# qr_app

A Flutter project for handling and reading QR codes.

## Requirements
- Flutter 3.24.5
- Melos - Monorepo management tool
- Pigeon - Flutter plugin generator

2 - Activate Melos
```bash
dart pub global activate melos 
```
3 - Run melos bootstrap
```bash
melos bootstrap
```
## Run pigeon Example
```bash
dart run pigeon --input FILE_PATH
```
